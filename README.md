# Авто Блокнот (AutoNote)

Мобильное Android-приложение для дипломной работы на тему:

> Разработка информационной системы для управления учетом сервисного обслуживания и мониторинга затрат на эксплуатацию автомобиля.

## Возможности MVP

- добавление автомобиля с маркой, моделью, годом, номером, пробегом и типом топлива;
- учет расходов по категориям: топливо, ТО, ремонт, запчасти, страховка, мойка, парковка, прочее;
- история затрат с суммой, пробегом, датой и комментарием;
- базовая аналитика: общая сумма затрат, стоимость на 1 км, структура расходов по категориям;
- сервисные напоминания по пробегу и сроку.
- backend-модуль с каталогом автомобилей, аккаунтами, токенами и хранением записей.

## Технологии

- Kotlin;
- Jetpack Compose;
- Material 3;
- Room;
- MVVM;
- Coroutines и Flow.

## Структура проекта

- `data/model` - сущности базы данных;
- `data/local` - Room database и DAO;
- `data/repository` - слой доступа к данным;
- `viewmodel` - состояние экранов и бизнес-логика;
- `ui` - Compose-интерфейс приложения.
- `backend` - Ktor API и SQLite-база серверной части.

## Как запустить

Откройте папку проекта в Android Studio, дождитесь Gradle Sync и запустите модуль `app` на эмуляторе или Android-устройстве.
